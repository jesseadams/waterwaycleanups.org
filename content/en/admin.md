# Admin Dashboard

This is the administrative interface for managing cleanup events and volunteers. Please log in with your admin credentials to access the management tools.

## Features

- **Event Management**: Create, edit, and delete cleanup events
- **Volunteer Management**: View volunteer profiles and RSVP history  
- **Analytics**: View participation statistics and generate reports
- **Data Export**: Export event and volunteer data for analysis

Access is restricted to authorized administrators only.
